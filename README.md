# Blender-phone-model
Модель Ipone 12 с зарядкой в Блендере.
## Теория
3d модели сделаны на основе  элементрных  функций Blender  и раскрашены с помощью шейдинга. На телефоне рисутствуют кнопки, разъем для зарядки, динамики.
